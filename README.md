NextStep - **A modern online store for sports sneakers**.

## Stack
| Frontend                       | Backend & BD                 | Instruments              |
|--------------------------------|------------------------------|--------------------------|
| Next.js 16.2 (App Router)      | PostgreSQL (Neon)            | TypeScript               |
| React 19.2 + Server Components | Prisma ORM                   | Tailwind CSS + Shadcn/ui |
| Zustand                        | Auth.js (Former NextAuth.js) | Zod                      |
| React Hook Form                | Server Actions               | ESLint + Prettier        |
